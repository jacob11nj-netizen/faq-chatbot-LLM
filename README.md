# FAQ Chatbot with LLM

## Description
A Python chatbot that answers FAQs using a Large Language Model (LLM). Demonstrates API calls, prompt engineering, and chatbot deployment.

## Tech Stack
- Python
- OpenAI API / HuggingFace Transformers

## Usage
1. Clone repo: `git clone https://github.com/yourusername/faq-chatbot-llm.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Set your API key in `config.py`
4. Run: `python chatbot.py`
